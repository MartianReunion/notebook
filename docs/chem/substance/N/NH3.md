# 氨气（$\ce{NH3}$）

## 1. 物理性质

!!! tip inline end "关于实验"

    参见《[$\ce{NH3}$的喷泉实验](../../experiment/NH3.md)》。

1. 无色、刺激性气味气体，$\rho_{\ce{NH3}}\geq\rho_{空气}$，有毒。
2. 易液化 $\because$ 沸点高，有氢键。
3. $\ce{NH3}$极易溶于水

!!! info "常见气体地水溶性"
    1. $\ce{CO2}$：1:1
    2. $\ce{Cl2}$：1:2
    3. $\ce{H2S}$：1:2.6
    4. $\ce{SO2}$：1:40
    5. $\ce{HCl}$：1:500
    6. $\ce{NH3}$：1:700

## 2. 化学性质

### 2.1. 与$\ce{H2O}$反应

$$\ce{NH3 + H2O <=>[大量] NH3·H2O <=>[少量] NH4+ + OH-}$$
