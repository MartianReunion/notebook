# $\ce{N_2}$

## 1. 性质

### 1.1. 物理性质

略

### 1.2. 化学性质

由于$\ce{N#N}$键能极大，$\ce{N_2}$常温常压下稳定。

???+ tip

    研究化学性质，可结合《[物质的分类](../../theory/物质的分类.md)》，进行枚举。

1. 与非金属单质反应

    $$\mathrm{N_2 + 3H_2 \xrightleftharpoons[\text{催化剂}]{\text{高温、高压}} 2NH_3}$$

    $$\mathrm{N_2} + \mathrm{O_2} \overset{\text{高温、放电}}{=\!=} 2\mathrm{NO}$$

2. 与活泼金属反应
   
    $$\ce{N2 + 3Mg = Mg3N2}$$

    $$\ce{N2 + 6Na -> 2Na3N}$$

## 2. 氮的固定

氮的固定是指，将游离态的氮元素转化为化合态的氮元素的过程。

方式：

1. 自然固氮：  
   a. 高能固氮：雷电放电，生成$\ce{NO}$  
   b. 生物固氮：豆科植物根瘤菌
2. 人工固氮：  
   a. 合成$\ce{NH3}$  
   b. 仿生固氮

关于自然界的氮元素，见必修二教材$P_{72}$，图5-9.
