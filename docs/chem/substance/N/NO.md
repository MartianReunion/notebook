# 氮的氧化物与酸

## 1. 类别

1. $\ce{N2O}$（+1价）：别称笑气，是麻醉剂，
2. $\ce{NO}$（+2价）：无色气体，有毒
3. $\ce{N2O3}$（+3价）：酸性氧化物，$\ce{HNO2}$的酸酐，有毒
4. $\ce{NO2}$（+4价）：红褐色，有毒
5. $\ce{N2O4}$（+4价）：无色，常用作火箭推进剂，有毒
6. $\ce{N2O5}$（+5价）：酸性氧化物，$\ce{HNO3}$的酸酐，有毒

## 2. 对比$\ce{NO}$与$\ce{NO2}$

| 物理性质 | $\ce{NO}$ | $\ce{NO2}$ |
| -------- | ---------- | ---------- |
| 颜色     | 无色       | 红褐色     |
| 气味     | 无味       | 刺激性气味 |
| 毒性     | ☠️有毒      | ☠️有毒      |
| 水溶性   | 能溶       | 易溶       |
| 沸点     | -          | 易液化     |

!!! warning inline end "酸性氧化物"

    请回顾《[物质的分类](../../theory/物质的分类.md)》，只生成一种盐和水的才叫做酸性氧化物。此处同时生成了硝酸和亚硝酸，并且发生了氧化还原反应，所以$\ce{NO2}$不是酸性氧化物。

化学性质：

1. 与$\ce{O2}$反应  
   $\ce{NO}$：$\ce{2NO + O2 = 2NO2}$  
   $\ce{NO2}$：不反应
2. 与$\ce{H2O}$反应  
   $\ce{NO}$：不反应  
   $\ce{NO2}$：$\ce{3NO2 + H2O = 2HNO3 + NO}$
3. 与碱反应  
   $\ce{NO}$：不反应  
   $\ce{NO2}$：  
   a. $\ce{2NO2 + 2NaOH = NaNO3 + NaNO2 + H2O}$  
   b. $\ce{NO + NO2 + 2NaOH = 2NaNO2 + H2O}$

特殊地,以下反应可以自发地进行：

$$\ce{2NO2 <=> N2O4}$$

这意味着，没有任何纯净的$\ce{NO2}$或$\ce{N2O4}$。

## 3. 雷雨发庄稼地原理

$$\ce{N2 ->[O2][放电] NO ->[O2] NO2 ->[H2O] HNO3 ->[\text{进一步反应}] 硝酸盐}$$
